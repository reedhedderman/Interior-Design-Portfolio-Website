Flexigrid: A Failsafe Grid System

Flexigrid is designed to be a safe grid system for first time programmers. Using CSS Grid, Flexigrid ensure everything you do follows the grid system you specify, even if you forget (or don't need!) containers.

↓

Features

Works with any HTML entity

The grid automagically places items on the grid. If you want four images in a row, then drop four ‹img› tags next to each other, or group things using ‹div›. Flexigrid works with the outermost object!

Designed for Safety

Too many objects for your row? Too few? No worries! Flexigrid uses template-columns to make sure objects sit and flow how they need.

Not-So-Features

Rigid

This code is a bit rigid and requires a lot of conditional rows to play safe. You're stuck with whatever layouts are written, and if you don't know how to add one (i.e. needed 1/8 styles) then you'd be out of luck. Not-so-flexigrid...

Too Safe?

Being too protective of the coder can create issues. Either proper coding isn't learned or the code gets in the way and doesn't allow for much creativity. Users end up building EOS'es (Every Other Site).

Thin

Just a grid. No padding. No spaces. Nothing special. Doesn't cover a lot of edge cases. This could change with time...

Okay that's nice... how do I use this?

Step 1: Load the script in your header

‹link rel="stylesheet" href="path/to/your/scripts/flexigrid.css"›

Step 2: Use a row class from above

‹div class="grid-third"›
    ...
‹/div›

Step 3: Add objects! Flexigrid will layout objects for you.

‹div class="grid-third"›
    ‹img src="an-image.png"›
    ‹div› ... ‹/div›
    ‹p› ... ‹/p›
‹/div›
